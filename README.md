# Credit Card Transaction Analysis

## Project Overview
This project analyzes credit card transaction data using Python, Pandas, and SQL.
It demonstrates a basic data engineering workflow including ETL, data cleaning,
and analytical queries.

## Tech Stack
- Python
- Pandas
- SQL (Oracle syntax)
- CSV

## Project Structure
credit_transaction_project/
│
├── data/
│   ├── transactions.csv
│   └── clean_transactions.csv
│
├── scripts/
│   ├── etl.py
│   └── analysis.py
│
└── sql/
    └── analysis.sql

## Key Features
- ETL pipeline using Pandas
- Data quality checks
- Credit vs Debit analysis
- Failed transaction percentage
- Customer-wise spending analysis
